CPNS 2024 Formation Analysis Dashboard

PROJECT OVERVIEW
Project analisis data komprehensif yang mengubah data mentah formasi CPNS (Calon Pegawai Negeri Sipil) 2024 dari SSCASN menjadi dashboard business intelligence interaktif. Project ini memberikan insights strategis untuk pelamar, instansi pemerintah, dan pembuat kebijakan dalam proses rekrutmen CPNS Indonesia.

Jenis Project: Data Analytics & Business Intelligence                                                                                                                          
Industri   :Pemerintahan & Sektor Publik                                                                                                          
Tools      :MySQL, DBeaver, Power BI                                                                                    

BUSSINES OBJECTIVE
Dashboard ini menjawab pertanyaan tentang rekrutmen CPNS 2024:
1. Distribusi Geografis - Formasi tersebar di mana saja?                                                                                                                   
2. Analisis Kompetisi - Formasi mana yang paling kompetitif?                                                                                   
3. Prospek Karir & Kompensasi - Posisi dan gaji seperti apa yang ditawarkan?                                                                                           
4. Persyaratan Pendidikan - Kualifikasi apa yang paling dicari?                                                                                       
5. Inklusivitas - Seberapa inklusif rekrutmen CPNS 2024                                                                                     

PROJECT TIMELINE                                                                                                                             
Fase 1: Pengaturan Basis Data & Persiapan Data                                                                                   
Fase 2: Analisis SQL & Pembersihan Data                                                                                                                    
Fase 3: Pengembangan Dasboard di Power BI                                                                                                                           
Fase 4: Implementasi Fitur Interaktif                                                                                                                         
Fase 5: Pemolesan & Validasi Akhir                                                                                       


KEY INSIGTH
1. Distribusi Formasi
   Total: 1,500 formasi dengan 25,000 kuota nasional
   Distribusi pendidikan: S1 dominan (52.33%), diikuti D3 dan SMA
   15% formasi menerima pelamar disabilitas
2.  Analisis Kompetisi
    Instansi dengan kuota terbanyak: Kementerian Agama, BRIN, Kemen PUPR
3.  Prosepek Karir & Kompensansi
    Posisi dengan gaji tertinggi   : -DOKTER PENDIDIK KLINIS SUB SPESIALIS BEDAH-BEDAH VASKULER
                                      DAN ENDO VASKULER
                                     -DOKTER SUB SPESIALIS PENYAKIT DALAM-REUMATOLOGI
                                     -DOKTER SUB SPESIALIS ANASTESI MANAJEMEN NYERI
4. Pattern Rekrutmen
   Pendidikan paling dicari: S1 (52.33%), D3 (15.33%), D4 (14.02%)
   Formasi inklusif: 0.70% terbuka untuk disabilitas


FITUR DASHBOARD
Visualizations:
   -Executive Summary (KPI Cards)
   -Total Formasi, Total Kuota, Avg Gaji Maximum, % Disabilitas
Analytical Charts
   -Top 10 Instansi by Kuota
   -Top Posisi by Gaji Tertinggi
   -Range Gaji per Pendidikan (Waterfall)
   -Distribusi Pendidikan (Pie Chart)
Interactive Elements
   -Filter by Lokasi (Dropdown)
   -Filter by Pendidikan (Multi-select)
   -Cross-filtering antara charts
   -Detailed Table untuk drill-down

STRATEGIC RECOMMENDATION
Untuk Pelamar:
   -Strategi Aplikasi: Fokus pada formasi dengan kuota "Banyak" untuk peluang lebih besar
   -Pilihan PendIdikan: S1 memberikan ROI tertinggi dalam hal kompensasi
   -Lokasi: Pertimbangan daerah dengan kompetisi lebih rendah
Untuk Instansi Pemerintah:
   -Inklusivitas: Tingkatkan formasi untuk disabilitas (baru 0.70%)
   -Distribusi: Perluas formasi ke daerah tertinggal
   -Kompetisi: Formasi dengan kuota kecil perlu evaluasi kebutuhan
